### 运行环境
node v20.11.1
pnpm 9.4.0

### 开始开发
```bash
npm install pnpm # 安装 pnpm

pnpm i # 安装依赖

pnpm dev # 开发命令

pnpm build # 构建命令

pnpm preview # 预览构建结果
```
